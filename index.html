Step 1 - Make sure your computer is ready for Visual Studio

About Console

Console window is very useful for interacting with and debugging your code. The console. log() is a function in JavaScript which is used to print any kind of variables defined before in it or to just print any message that needs to be displayed to the user. You can open the console window by pressing ctrl+shift+j in chrome and ctr+shift+k in firefox. In console window you can type your javascript code.You should use the console. log() method to print to console JavaScript.
